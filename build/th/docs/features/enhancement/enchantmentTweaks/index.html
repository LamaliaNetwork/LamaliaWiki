Enchantments Guide

How to get overloaded enchantments

วิธีได้เอนแชนต์เมนต์ที่เกินระดับ
ให้รวมเอนแชนต์เมนต์สองอันที่มีระดับเดียวกันเพื่อให้ได้ระดับถัดไป [ 4+4 = 5, 5+5 = 6, ... ]
ค่าใช้จ่ายจะไม่แพงมากเกินไป!
