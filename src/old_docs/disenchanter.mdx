# Disenchanter

Disenchanter ช่วยให้คุณถอด enchant ออกจากไอเทม แต่มีความเสี่ยงและค่าใช้จ่าย

  description="The Disenchanter allows you to extract enchantments from items, but it comes with risks and costs."

/>

## Basic Mechanics

:::info Scavenge Cost
ค่าใช้จ่ายในการ Scavenge
การ scavenge มีค่าใช้จ่ายคงที่
:::

## How to Scavenge

:::tip Scavenging Process
ขั้นตอนการ Scavenge
1. **เตรียมไอเทม**
   - ตรวจสอบความทนทานและ enchantment ของไอเทม

2. **เริ่มการ Scavenge**
   - ใช้คำสั่ง scavenge หรือใช้งานที่ scavenging station

3. **ดำเนินการ Scavenge**
   - จ่ายค่าใช้จ่ายที่กำหนด
   - พยายามถอด enchantment
:::
