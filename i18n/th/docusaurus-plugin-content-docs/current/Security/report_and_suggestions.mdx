# 📢 Feedback and Bug Reporting

ยินดีต้อนรับสู่ระบบรายงานข้อบกพร่อง และข้อเสนอแนะ! ระบบนี้ได้รับการออกแบบมาเพื่อช่วยให้รายงานข้อบกพร่อง และเสนอแนะแนวคิดสำหรับโครงการ
ซึ่งจะช่วยให้เราปรับปรุงโครงการและทำให้ดียิ่งขึ้นสำหรับทุกคน ส่วนนี้อยู่ใน Discord

## Available Commands (for discord)
คำสั่งหลัก:
- `/bug` - ใช้สิ่งนี้ในช่อง #bug
- `/idea` - ใช้สิ่งนี้ในช่อง #suggestions

## Command Format
`/<Command> <title> <attachments>`
ตัวอย่าง: `/bug Item Duplication Glitch [screenshot.png]`

## After Creating a Report
- จะสร้างกระทู้สำหรับการสนทนา
- ผู้ใช้สามารถแสดงความเห็นด้วยอีโมจิเพื่อโหวตรายงาน

## Suggestions
ข้อเสนอแนะต้องได้รับการโหวตขั้นต่ำ 10 ครั้งเพื่อพิจารณาสำหรับการพัฒนา (สร้างการ์ดสำหรับนักพัฒนา)

## Karma System
คุณสามารถรับ Karma จากการมีส่วนร่วมของคุณได้:
- 1 Karma สำหรับแต่ละการโหวตให้กับไอเดียของคุณ
- 10 Karma เมื่อรายงาน `/bug` ของคุณถูกทำเครื่องหมายว่าเสร็จสมบูรณ์
- 20 Karma เมื่อรายงาน `/idea` ของคุณถูกทำเครื่องหมายว่าเสร็จสมบูรณ์

*ผลประโยชน์ในอนาคตของ Karma จะประกาศให้ทราบในภายหลัง โปรดติดตาม!*

## Tips
- รายงานข้อบกพร่องและแนวคิดของคุณให้ชัดเจน และกระชับ
- ใช้ช่องทางที่เหมาะสมสำหรับแต่ละคำสั่ง
- เข้าร่วมการอภิปรายเพื่อปรับปรุงแนวคิดและช่วยแก้ไขข้อบกพร่อง
- โหวต 👍 การสนับสนุนอันมีค่าจากผู้อื่น
